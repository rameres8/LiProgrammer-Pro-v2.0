# LiProgrammer-Pro-v2.0
🧠 LiProgrammer Pro v2.0 by LiiS8

<img width="500" height="500" alt="app_icon" src="https://github.com/user-attachments/assets/f5ebaa44-e69f-417f-aff1-713ff353af6c" />




LiProgrammer Pro v2.0 is an advanced professional software built with Qt6, designed to work with CH341A / CH341X programmers through a modern graphical interface based on a Glassmorphism design.

It has been developed to deliver the highest level of speed, stability, and precision when working with various memory chips.

⚡ Supported Memory Types

The software supports a wide range of memory chips:

EEPROM
SPI Flash
I²C Memory
And other compatible chip types
✨ Core Features
✔️ Read / Write / Erase full chip operations
✔️ High performance and optimized processing speed
✔️ Support for hundreds of chip types
✔️ OTP / Register Sections support
✔️ Modern graphical interface (Glass UI)
✔️ Advanced chip search engine
✔️ Built-in driver management and installation
✔️ Full compatibility with Windows 10 / 11 (64-bit)
✔️ Smooth and optimized user experience in v2.0
🚀 v2.0 Release Highlights
🔎 Smart Search System

The chip database search system has been improved and now includes:

Search by typing the first letters of the chip name
Double-click on Search Chip to open an advanced window:
Full list of supported chips
Detailed technical information for each chip
Fast sorting and smart filtering system
🧱 New Technical Improvements
Added Block Map Viewer
Added Chip Statistics module
Added display of the currently connected driver in the interface
Improved overall performance and stability
Fixed multiple internal bugs (Bug Fixes)
Added a future compatibility layer for WCH Driver support
to avoid conflicts with the original WCH DLL driver
🔧 Driver Management System

You can now:

Install drivers directly from inside the software
Supported drivers:
✔ libusbK (Recommended)
✔ libusb-win32
❌ WDM (currently unstable, will be improved in future updates)

📌 Recommendation: Use libusbK for the best stability and performance.

🧰 Technologies Used
Qt 6 Framework
C++ (High Performance Core)
Windows API Integration
Supported Operating Systems:
Windows 10 (64-bit)
Windows 11 (64-bit)
